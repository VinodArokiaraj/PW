# Playwright Automation Framework

A scalable end-to-end UI automation framework built using Playwright with JavaScript.

## Installation
Clone the below repository 
git clone https://github.com/VinodArokiaraj/PW.git

## 🚀 Tech Stack

- Playwright
- JavaScript (Node.js)
- Page Object Model (POM)
- Allure Reporting
- GitHub Actions (optional CI integration)
